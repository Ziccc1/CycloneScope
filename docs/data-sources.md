# 必要数据来源

| 数据 | 链接 | 用途 |
|---|---|---|
| NOAA IBTrACS | https://www.ncei.noaa.gov/products/international-best-track-archive | 全球气旋轨迹和强度 |
| Copernicus ERA5 | https://cds.climate.copernicus.eu/datasets/reanalysis-era5-single-levels?tab=overview | 真实 u10/v10 风场 |
| EM-DAT | https://doc.emdat.be/docs/data-structure-and-content/emdat-public-table/ | 报告死亡、受灾人数和经济损失 |
| TCE-DAT | https://dataservices.gfz-potsdam.de/pik/showshort.php?id=escidoc%3A2770889 | 历史气旋人口和资产暴露 |
| WorldPop | https://hub.worldpop.org/ | 全球人口暴露栅格 |
| Natural Earth | https://www.naturalearthdata.com/downloads/ | 全球底图和城市 |
| 台湾避难收容处所 | https://data.gov.tw/dataset/73242 | 设施位置和容量 |
| 台湾收容处所开设情况 | https://data.gov.tw/dataset/12849 | 动态开设状态，可选 |
| 台湾救援与应变单位 | https://data.gov.tw/dataset/5969 | 消防、救援和应变中心 |
| 台湾医疗院所查询 | https://data.gov.tw/dataset/139250 | 医疗设施点位 |
| 台湾统计区人口 | https://data.gov.tw/dataset/18681 | 人口暴露和设施需求 |
| 台湾最小统计区图 | https://data.gov.tw/dataset/25128 | 人口空间连接 |
| 台湾县市界线 | https://data.gov.tw/dataset/7442 | 县市筛选和汇总 |
| OSM Taiwan Extract | https://download.geofabrik.de/asia/taiwan.html | 道路网络和点位补充 |

所有下载结果必须记录来源、下载日期、许可证、原始字段和清洗脚本版本。EM-DAT 的访问与再分发条件需要在纳入仓库前单独核对。
